# To-do List

A simple To-do list with a dark blue themed, helps you to list your tasks, goals and daily routine with a clean and focused interface.

## 🚀 Features

- Add new tasks to your list
- Delete individual tasks
- Clean and focused dark blue themed interface with glassmorphism effects
- Responsive design that works on different screen sizes
- Minimalist UI with smooth animations and hover effects

## 🔧 How It Works

This is a client-side web application built with HTML, CSS, and JavaScript. The application allows users to:

1. Enter a task in the input field
2. Click the "Add" button to add the task to the list
3. View all tasks in an ordered list
4. Delete any task by clicking the "Delete" button next to it

The tasks are stored temporarily in the browser's memory and will be lost when the page is refreshed.

## 🏃 How to Run

1. Download or clone this repository
2. Navigate to the `To-do List` folder
3. Open `index.html` in your web browser
4. Start adding and managing your tasks!

## 📝 Notes

- This application does not persist data - tasks are only stored in memory and will disappear on page refresh
- No backend or database is required; it's a pure frontend application
- Compatible with modern web browsers
- Built with vanilla JavaScript, no external dependencies required
